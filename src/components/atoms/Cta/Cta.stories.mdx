## Overview

Depending on the type of CTA required, you may decide to use a link over a solid or an outlined cta over a solid cta.

```tsx
import { Cta } from "../components/Cta/Cta";

export const ExamplePage () => {
  return (
    <>
      ...
      <Cta variant="solid" href="#" label="Get in touch" />
    </>
  );
}
```
